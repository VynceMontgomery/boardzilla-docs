# Installation

1. Create an starter game template by running the following command. It will guide you through setting up your game.

```console
npx @boardzilla/create-game
```

2. Change into your newly created directory using `cd [game-name]`.
3. Run `npm run dev` to start development.
4. Go to [http://localhost:8080](http://localhost:8080) to start developing.
